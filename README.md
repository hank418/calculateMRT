# calculateMRT
2/1北捷取消8折優惠，到底哪個划算？ 1280? / 忠誠度?

工作天數計算依[政府資料開放平臺/政府行政機關辦公日曆表](https://data.gov.tw/dataset/26557)為準


## Usage
```shell
$ python3 main.py
```
